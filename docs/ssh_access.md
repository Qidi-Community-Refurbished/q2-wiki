# SSH Access

Being able to SSH into a machine is useful to flash firmware, modify files from the terminal, and download and run scripts

Use a command line interface, like Windows terminal or [Tabby](tabby.sh) to ssh into your printer.

`ssh user@your_ip_adress` 

There are two user and password combos as of 01.01.02.01 firmware update. This update and all after have the option of both users, while 1.1.1 and down only have `mks`

| User | Password |
|______|__________|
|mks   | makerbase|
|qidi  | qiditech |
